# Responsive Portfolio Website

## 📌 Project Overview
This is a personal portfolio website built using HTML5 and CSS3. It showcases my skills, projects, and contact information.

---

## 🚀 Features

### Task 1:
- Semantic HTML structure
- Multiple pages (Home, About, Projects, Contact)
- Navigation menu

### Task 2 (Advanced CSS3 & Responsive Design):
- CSS Grid for layout structure
- Flexbox for navigation alignment
- Mobile-first responsive design using media queries
- CSS variables for theme management
- Card-based UI design

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- CSS Grid
- Flexbox
- Media Queries

---

## 📱 Responsive Design
The website is fully responsive and works on:
- Mobile devices
- Tablets
- Desktop screens
